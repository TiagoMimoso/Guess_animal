# Animal Guess

A simple game where you try to guess animals based on clues provided by an AI.

## Requirements

Before using this project, you need to have:
- **Ollama** installed
- **qwen2.5:14b** model

Make sure Ollama is running and the qwen2.5:14b model is downloaded before starting the game.
